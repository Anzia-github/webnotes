三类script
![[Pasted image 20220414113047.png]]

- script
	- 浏览器在解析HTML的时候
	- 遇到一个没有任何属性的script标签
	- 就会暂停解析
	- 先发送网络请求获取该JS脚本的代码内容
	- 然后让JS引擎执行该代码
	- 代码执行完毕恢复解析
	- ![[Pasted image 20220414113324.png]]

- 痛点
	- script阻塞了HTML的解析
	- 如果获取JS脚本的网络请求迟迟得不到响应
	- 或JS脚本的执行时间过长
	- 会导致白屏，用户看不到东西

- 解决
	- [[async]]属性
	- [[defer]]属性
